Mengenal Semantic Elements Di HTML5

  • Article
    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita, tag artikel bisa kita gunakan dengan elemen header dan footer. Karena tag artikel memiliki type block, artinya elemen ini memiliki lebar sesuai lebar browser jika kitaa tidak menentukan lebarnya
  • Aside
    • Elemen ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artike. Karena aside memiliki type/sifat block maka lebarnya sesuai page jika tidak menentukan lebarnya
